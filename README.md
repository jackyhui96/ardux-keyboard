# ardux-keyboard

## Changes from the 40p ardux
* Miryoku instead of 40p ardux ansi
    * MIRYOKU_CLIPBOARD=WIN
    * MIRYOKU_NAV=VI
    * Doesn't include additional features of `Boot`, `Tap`, `Extra`, `Base`, `Opp`, `Cur` 
* UK keycodes
    * `KC_BSLS -> KC_NUBS`
    * `KC_HASH -> KC_NUHS`
    * `KC_PIPE -> S(KC_NUBS)`
